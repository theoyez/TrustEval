Transparent evaluation for AI systems. Scores are Accuracy, Grounding, Safety, and a Trust blend.

Run ?Run identifier. Model ?Model or system name. Accuracy ?Partial-credit match vs expected (exact / token overlap / regex / edit). Grounding ?Citations must be allowed (Per-Item → Domain). If cited source isn’t in retrieved top‑k, score is capped. Safety ?Regex packs for PII/Toxicity/Policy apply bounded deductions. Trust ?0.6×Accuracy + 0.3×Grounding + 0.1×Safety. Audit ?Quick RAG audit: top‑k retrieved, cited keys, and consistency. Report
