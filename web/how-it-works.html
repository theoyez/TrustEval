← Leaderboard—How it works

TrustEval scores Accuracy, Grounding, Safety, and a weighted Trust composite. It also surfaces a RAG Audit Trail and optional Merkle proofs for cited chunks.

  1. Run your system on our prompts.
  2. Write data/runs/<run_id>.jsonl with fields: id, output, retrieved (top‑k with scores), optional proof (chunk hash + doc root).
  3. Build locally: python3 scripts/build.py to generate metrics + HTML reports; publish static files only.
