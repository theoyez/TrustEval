TrustEval scores Accuracy, Grounding, Safety, and a weighted Trust composite. Reports include a RAG Audit Trail and optional Merkle proof status for cited chunks.

  1. Run your system on our prompts.
  2. Write data/runs/<run_id>.jsonl with fields: id, output, retrieved (top‑k with scores), optional proof (chunk hash + doc root).
  3. Build locally (or just publish artifacts): generate metrics + HTML and host as static files.
