How it works

TrustEval scores model outputs for Accuracy (partial credit), Grounding (Per-Item allowlist ▶ Domain allowlist ▶ RAG consistency), Safety (regex packs), and combines them into a Trust score. Strict sources include Merkle proofs.

  1. Per-Item allowlist: cite a listed source (IDs/URLs) for full grounding credit.
  2. Domain allowlist: fall back to trusted AI research domains.
  3. RAG consistency: if a run provides retrieved, at least one citation must be in that set.
