← Leaderboard—How it works

TrustEval computes Accuracy, Grounding, and Safety, plus a weighted Trust. Reports include a RAG Audit Trail and optional Merkle proof placeholders for cited chunks.

  1. Run your system on our prompts.
  2. Write data/runs/<run_id>.jsonl with fields: id, output, retrieved (top-k with scores), optional proof (chunk hash + doc root).
  3. Publish the artifacts (metrics + HTML) as static files.
